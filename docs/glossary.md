# Глоссарий

🚧 **В разработке.**

Единые определения терминов, на которые ссылаются другие страницы руководства.
Каждый термин определяется здесь один раз; остальные страницы дают ссылку сюда.

Предполагаемое наполнение:

- Термины RAG (ingestion, retrieval, generation).
- Термины агентных систем (tool use, планирование, мультиагентность).
- Сквозные термины (evaluation, guardrails, observability).
