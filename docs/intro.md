# Enterprise RAG & Agents Handbook

🚧 **В разработке.** Это заглушка — содержание будет наполняться отдельно.

Практическое, основанное на первых принципах руководство по production-системам
RAG (retrieval-augmented generation) и агентным системам.

## Для кого это руководство

Материал написан сразу для трёх аудиторий:

- **Изучающие тему** — любой инженер, который хочет разобраться в RAG и агентах с нуля,
  включая «почему» и разбор режимов отказа, а не только перечень возможностей.
- **Личный справочник автора** — долговременная запись принципов проектирования и решений.
- **Портфолио** — демонстрация зрелости инженерной практики (eval, guardrails,
  observability, дисциплина проектирования).

## Структура

- **Часть I — RAG:** ingestion, retrieval, generation и сквозные аспекты.
- **Часть II — Агенты:** агентный RAG, использование инструментов, циклы, планирование, мультиагентность.
- **Глоссарий:** единые определения терминов, на которые ссылаются другие страницы.
